// formule avec barre de fraction
TITRE : Loi des fractions en algèbre
FORMULE: (a+b)/(c+d) = (a/c) + (b/d) sous certaines conditions
DESCRIPTION : Cette formule montre comment décomposer une fraction complexe en somme de fractions simples.

// formule avec exposants
TITRE : Identité remarquable du carré
FORMULE: (a+b)² = a² + 2ab + b²
DESCRIPTION : Formule fondamentale pour développer le carré d'une somme.

// formule avec multiplication
TITRE : Produit de deux sommes
FORMULE: (a+b)(c+d) = ac + ad + bc + bd
DESCRIPTION : Développement du produit de deux binômes selon la double distributivité.

// formule avec racine carrée
TITRE : Racine carrée d'un produit
FORMULE: √(ab) = √a × √b
DESCRIPTION : Propriété de la racine carrée sur un produit.

// formule avec intégrale
TITRE : Intégrale d'une puissance
FORMULE: ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C (pour n ≠ -1)
DESCRIPTION : Formule fondamentale pour l'intégration des fonctions puissance.

// formule avec somme
TITRE : Somme des n premiers entiers
FORMULE: ∑_{k=1}^{n} k = n(n+1)/2
DESCRIPTION : Formule pour calculer la somme des n premiers nombres entiers.

// formule avec limites
TITRE : Limite remarquable exponentielle
FORMULE: lim_{x→0} (eˣ - 1)/x = 1
DESCRIPTION : Limite fondamentale pour la fonction exponentielle.

// formule avec dérivée
TITRE : Dérivée d'une puissance
FORMULE: d/dx (xⁿ) = n xⁿ⁻¹
DESCRIPTION : Règle de dérivation des fonctions puissance.

// formule avec fraction complexe
TITRE : Simplification de fraction
FORMULE: (a/b)/(c/d) = (a×d)/(b×c)
DESCRIPTION : Règle pour diviser deux fractions.

// formule avec équation quadratique
TITRE : Solutions d'une équation quadratique
FORMULE: ax² + bx + c = 0 ⇒ x = [-b ± √(b² - 4ac)]/(2a)
DESCRIPTION : Formule quadratique pour résoudre les équations du second degré.

// formule avec logarithme
TITRE : Produit de logarithmes
FORMULE: log(ab) = log a + log b
DESCRIPTION : Propriété fondamentale des logarithmes.

// formule avec exponentielle
TITRE : Produit d'exponentielles
FORMULE: eˣ × eʸ = eˣ⁺ʸ
DESCRIPTION : Règle pour multiplier des exponentielles de même base.

// formule avec trigonométrie
TITRE : Identité trigonométrique fondamentale
FORMULE: sin² θ + cos² θ = 1
DESCRIPTION : Relation fondamentale entre sinus et cosinus.

// formule avec dérivée de produit
TITRE : Dérivée d'un produit
FORMULE: d/dx (uv) = u'v + uv'
DESCRIPTION : Règle du produit pour la dérivation.

// formule avec intégrale définie
TITRE : Intégrale définie d'une constante
FORMULE: ∫_a^b k dx = k(b - a)
DESCRIPTION : Formule pour l'intégrale d'une fonction constante.

// formule avec factorisation
TITRE : Différence de carrés
FORMULE: a² - b² = (a - b)(a + b)
DESCRIPTION : Formule de factorisation pour la différence de deux carrés.

// formule avec puissance de puissance
TITRE : Règle des puissances
FORMULE: (aᵐ)ⁿ = aᵐⁿ
DESCRIPTION : Règle pour élever une puissance à une autre puissance.

// formule avec racine n-ième
TITRE : Racine n-ième d'un produit
FORMULE: ⁿ√(ab) = ⁿ√a × ⁿ√b
DESCRIPTION : Propriété de la racine n-ième.

// formule avec série géométrique
TITRE : Somme d'une série géométrique
FORMULE: ∑_{k=0}^{n-1} rᵏ = (1 - rⁿ)/(1 - r) pour r ≠ 1
DESCRIPTION : Formule pour la somme des n premiers termes d'une suite géométrique.

// formule avec dérivée de quotient
TITRE : Dérivée d'un quotient
FORMULE: d/dx (u/v) = (u'v - uv')/v²
DESCRIPTION : Règle du quotient pour la dérivation.

// formule avec intégration par parties
TITRE : Intégration par parties
FORMULE: ∫ u dv = uv - ∫ v du
DESCRIPTION : Formule fondamentale pour l'intégration par parties.

// formule avec nombre complexe
TITRE : Module d'un nombre complexe
FORMULE: |a + bi| = √(a² + b²)
DESCRIPTION : Calcul du module d'un nombre complexe.

// formule avec dérivée trigonométrique
TITRE : Dérivée du sinus
FORMULE: d/dx (sin x) = cos x
DESCRIPTION : Dérivée de la fonction sinus.

// formule avec matrice
TITRE : Déterminant 2×2
FORMULE: det[[a, b], [c, d]] = ad - bc
DESCRIPTION : Calcul du déterminant d'une matrice 2×2.

// formule avec limite trigonométrique
TITRE : Limite trigonométrique fondamentale
FORMULE: lim_{x→0} (sin x)/x = 1
DESCRIPTION : Limite fondamentale pour la fonction sinus.
